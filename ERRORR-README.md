# راهنمای استفاده از بخش محصولات ERRORR

## نحوه اضافه کردن عکس و ویدیو

### روش 1: استفاده از اسکریپت Python (پیشنهادی)

1. فایل‌های عکس و ویدیو خود را در فولدر `images/errorr-products` قرار دهید.

2. اسکریپت Python را اجرا کنید:
   ```bash
   python update_errorr_media.py
   ```

3. این اسکریپت به صورت خودکار تمام فایل‌های عکس و ویدیو را پیدا می‌کند و لیست آن‌ها را در فایل `errorr-media.json` ذخیره می‌کند.

### روش 2: ویرایش دستی فایل JSON

می‌توانید فایل `errorr-media.json` را به صورت دستی ویرایش کنید:

```json
{
  "files": [
    "images/errorr-products/photo1.jpg",
    "images/errorr-products/video1.mp4",
    "images/errorr-products/photo2.webp"
  ]
}
```
