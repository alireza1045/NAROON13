# Naroon | وب‌سایت تابلوسازی حرفه‌ای

یک وب‌سایت مدرن و زیبا برای ارائه خدمات تابلوسازی با طراحی واکنش‌گرا و رابط کاربری جذاب.

## ویژگی‌ها

- **طراحی واکنش‌گرا**: سازگار با تمام دستگاه‌ها (دسکتاپ، تبلت، موبایل)
- **رابط کاربری مدرن**: طراحی تمیز و حرفه‌ای با انیمیشن‌های نرم
- **ناوبری روان**: اسکرول نرم با هایلایت لینک‌های فعال
- **گالری تصاویر**: نمایش نمونه کارها با فیلتر دسته‌بندی
- **فرم تماس**: فرم درخواست مشاوره و استعلام قیمت
- **انیمیشن‌های جذاب**: المان‌ها با اسکرول صفحه ظاهر می‌شوند
- **منوی موبایل**: منوی همبرگری برای دستگاه‌های موبایل
- **محتوای فارسی**: تمام محتوا به زبان فارسی و راست‌چین

## بخش‌های سایت

1. **بخش اصلی (Hero)**: پیام خوش‌آمدگویی با آمار و دکمه‌های فراخوان
2. **خدمات**: معرفی انواع خدمات تابلوسازی
3. **گالری**: نمایش نمونه کارها با امکان فیلتر
4. **درباره ما**: معرفی شرکت و ویژگی‌های برتر
5. **فرآیند کار**: مراحل انجام پروژه از ابتدا تا پایان
6. **تماس با ما**: اطلاعات تماس و فرم درخواست
7. **فوتر**: لینک‌های سریع و اطلاعات تماس

## خدمات ارائه شده

1. **حروف چنلیوم**: ساخت حروف چنلیوم با ظاهر لوکس و براق
2. **حروف پلاستیک**: ساخت حروف پلاستیکی با وزن سبک و قیمت مناسب
3. **حروف استیل**: ساخت حروف استیل با کیفیت بالا و ظاهر مدرن
4. **حروف فلزی**: ساخت حروف فلزی با استحکام بالا و مقاومت عالی
5. **تابلو نئون پلکسی**: طراحی و ساخت تابلوهای نئون پلکسی با نورپردازی زیبا
6. **تابلوی LED**: تابلوهای LED با مصرف انرژی پایین و روشنایی بالا
7. **انواع لایت باکس و باکس**: ساخت انواع لایت باکس با نورپردازی یکنواخت
8. **کارهای نمایشگاهی**: طراحی و ساخت استندها و المان‌های نمایشگاهی
9. **سازه های پلیگان 3D**: طراحی و ساخت سازه‌های پلیگان سه بعدی
10. **تابلو اروپایی**: طراحی و ساخت تابلوهای اروپایی با سبک کلاسیک

## ساختار فایل‌ها

```
website/
├── index.html          # فایل HTML اصلی با تمام بخش‌ها
├── styles.css          # استایل‌های CSS و طراحی واکنش‌گرا
├── script.js           # عملکردهای JavaScript
└── README.md           # این فایل
```

## نحوه استفاده

1. **باز کردن سایت**: فایل `index.html` را در مرورگر باز کنید
2. **ناوبری**: از منوی ناوبری برای جابجایی بین بخش‌ها استفاده کنید
3. **گالری**: از دکمه‌های فیلتر برای مشاهده نمونه کارها استفاده کنید
4. **تماس**: فرم تماس را پر کرده و درخواست خود را ارسال کنید
5. **موبایل**: سایت به صورت خودکار با دستگاه‌های موبایل سازگار می‌شود
