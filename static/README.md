# Static Files Directory

This directory contains all static files for the NAROON website.

## Structure

```
static/
├── css/
│   └── styles.css          # Main stylesheet
├── js/
│   ├── script.js           # Main JavaScript file
│   ├── ERRORR.js          # ERRORR products gallery JavaScript
│   └── gallery-page.js     # Gallery page JavaScript
└── data/
    ├── gallery-data.json   # Gallery manifest
    └── errorr-media.json    # ERRORR products media manifest
```

## Notes

- All CSS files are in `css/` directory
- All JavaScript files are in `js/` directory
- All JSON data files are in `data/` directory
- Images and videos are served from root `images/` and `videos/` directories
